# AI Handwritten Digit Recognizer

This project is a simple deep learning application that recognizes handwritten digits using the MNIST dataset.

## Technologies Used

- Python
- TensorFlow
- Keras
- NumPy
- Matplotlib

## Features

- Trains a neural network on handwritten digit images
- Predicts digits from 0 to 9
- Displays the predicted digit and actual digit
- Visualizes the test image

## How to Run

Install the required libraries:

```bash
pip install tensorflow matplotlib numpy
```

Run the project:

```bash
python app.py
```
